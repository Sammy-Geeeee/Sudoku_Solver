# Sudoku_Solver
Program to solve a simple 9x9 Sudoku board.

This is my implementation of a coding challenge I had read about.
This will only work for 9x9 Sudoku's and also only works in the console, no GUI.

To use to solve your own Sudoku's you will need to edit one of the existing Sudoku dictionaries within the program or you can copy/paste and edit another one.
